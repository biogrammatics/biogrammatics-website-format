- This overlay assumes you created your app with Tailwind: `rails new myapp -c tailwind`.
- If Action Mailer delivery isn't configured in development, add something like:
  # config/environments/development.rb
  config.action_mailer.delivery_method = :smtp
  config.action_mailer.smtp_settings = { address: 'localhost', port: 1025 }
  # and run `mailcatcher` or `letter_opener` for testing.
